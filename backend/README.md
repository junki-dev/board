# BOARD BACKEND

## 실행 명령어

### `yarn install`

- 패키지 설치

### `yarn start:[local|dev]`

- Backend Server 실행
- Backend URL : http://localhost:3000
- GraphQL Playground : http://localhost:3000/graphql

### `yarn build`

- [production] 빌드

### `yarn start`

- [production] Backend Server 실행

## 배포

### Dockerizing 명령어 예

```bash
# board:1.0 이미지 생성
$ docker build -t board:1.0 .
```
